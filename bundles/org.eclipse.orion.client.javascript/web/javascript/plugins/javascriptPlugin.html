JavaScript Tools Support

This plugin provides a variety of JavaScript tooling, including:

  • Editing - an editor with syntax highlighting, navigation, code completion, etc.
  • Navigation - source navigation from the editor as well as a variety of outines.
  • Content Assist - smart content assist while editing.
  • Search - named item searching, editor occurrences, etc.
