AST Manager Unit Tests

The tests cover the orion.edit.ASTManager. The following tests are defined:

  • test_getAST - tests asking the manager for an AST of a test content type text/foo
  • test_getAST_options - tests asking the manager for options used to create an AST for a test content type text/foo
  • test_AST_cache_is_used - tests that the AST re-asked for is the cached copy
  • test_AST_cache_is_invalidated - tests that the cache of ASTs is properly cleaned on changed events and input loads
